# Resources

Downloadable assets for the Atmospheric Depth Gallery project.

## Videos

Three recording formats are included:

| File        | Format         | Description                                             |
| ----------- | -------------- | ------------------------------------------------------- |
| `demo.mp4`  | 16:9 landscape | Clean rendering of the experience, no background effect |
| `stage.mp4` | 16:9 landscape | Blurred backdrop fill behind the experience             |
| `reel.mp4`  | 9:16 portrait  | Vertical format with blurred backdrop                   |

## Screenshots

Static captures of the experience.

## How to download

Click any file → **Download raw file** button (top right on GitHub).
